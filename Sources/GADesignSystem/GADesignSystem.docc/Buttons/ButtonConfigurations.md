# ButtonConfiguration

Use this configurations with the ``GABaseButtonStyle``

## Overview

Base Buttons can be configured in size (``GAButtonSize/small-enum.case``, ``GAButtonSize/regular``)

## Topics

### Button Style
- ``GAButtonStyle/pill``
- ``GAButtonStyle/roundedCorners``
- ``GAButtonStyle/square``

### Button Size
- ``GAButtonSize/small``
- ``GAButtonSize/regular``
- ``GAButtonSize/large``

### Button Type
- ``GAButtonType/primary``
- ``GAButtonType/secondary``
- ``GAButtonType/tertiary``

### Configuration Types
- ``GAButtonSize``
- ``GAButtonType``
- ``GAButtonStyle``
